## Metodo de uso  
 En caso de querer aser uso de este materia ya sea para aprendisaje o como retroalimentacion, el docigo se puede ejecutar facilmente desde la raiz (una vez teniendo los archivos en la maquina local), basta con introducir el comando ```dotnet build; dotnet run``` en la terminal, esto con el fin de primero revisar si todo funciona dentro del codigo y luego poder correro sin mayor dificultad.
 en su defecto, se pueden ejecutar de manera separada en su respectivo orden:
 - [ ] dotnet build
 - [ ] dotnet run
